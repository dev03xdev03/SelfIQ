# Sound Files

## Typewriter Sound

Füge hier eine `typewriter.mp3` Datei hinzu für den Tipp-Sound-Effekt.

Du kannst kostenlose Typewriter-Sounds von Seiten wie:

- https://freesound.org/
- https://pixabay.com/sound-effects/

Suche nach "typewriter" oder "keyboard typing"

Empfohlene Eigenschaften:

- Format: MP3
- Länge: Sehr kurz (0.1-0.2 Sekunden)
- Lautstärke: Moderat
- Typ: Einzelner Tastendruck-Sound

Sobald du die Datei hinzugefügt hast, aktiviere den Code in `StoryScreen.tsx` (entferne die Kommentare bei der Sound-Laden-Funktion).
